### 🚀 Projects

#### 🛡️ UPI Fraud Shield AI — Java Spring Boot Microservices
Production-grade Spring Boot backend that scores UPI transactions for fraud in real time using a 10-rule engine, pgvector RAG over RBI/NPCI circulars, and Gemini 2.5 Flash. Returns a fraud score, a plain-English explanation, and the exact regulatory clause that flagged it.

`Java` `Spring Boot` `Spring AI` `Gemini AI` `pgvector` `RAG` `Microservices` `PostgreSQL` `Apache Kafka` `Docker`

🔗 [View Repo](https://github.com/Priyanshu-Gupta-24/upi-fraud-shield-ai)

---

#### 🛡️ UPI Fraud Shield AI — Python FastAPI
Same fraud detection system rebuilt in Python with FastAPI and LangChain. Real-time scoring using a rules engine, RAG over RBI/NPCI guidelines, and Gemini LLM explanations.

`Python` `FastAPI` `LangChain` `Gemini AI` `pgvector` `RAG` `PostgreSQL` `Apache Kafka` `Docker`

🔗 [View Repo](https://github.com/Priyanshu-Gupta-24/upi-fraud-shield-ai-python)

#### 🏨 SmartStay AI — Dynamic Pricing for Indian Hotels
ML + GenAI pricing assistant for Indian hotels and homestays. XGBoost and RandomForest predict optimal room rates in ₹ based on occupancy, month, guest count, and competitor prices. Gemini AI explains each recommendation with demand drivers and revenue impact.

`Python` `FastAPI` `XGBoost` `Gemini AI` `React` `PostgreSQL` `Docker`

🔗 [View Repo](https://github.com/Priyanshu-Gupta-24/smartstay-ai)

#### 📈 Gold Price Prediction — Indian Market
Hybrid recursive forecasting system using Random Forest and XGBoost on 10 years of Indian market data. Predicts gold prices at 10/30/90-day horizons with state-wise INR/10g breakdowns across all Indian states.

`Python` `XGBoost` `Random Forest` `Time Series` `Pandas` `Jupyter`

🔗 [View Repo](https://github.com/Priyanshu-Gupta-24/Gold-Price-Prediction-ML)
